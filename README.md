# SEWE💳

SEWE est une mini application Flutter développée dans le cadre d’un test technique.
Elle permet de simuler une authentification utilisateur et l’affichage de transactions financières.

## 📱 Fonctionnalités
- Écran de connexion avec validation email et mot de passe sécurisé
- Liste de transactions avec animation fluide
- Page de détails d’une transaction
- Interface moderne avec dégradés et Material Design 3

## 🛠️ Technologies
- Flutter 3.38.4
- Dart
- Material Design

## 🔐 Sécurité
- Validation des champs utilisateur
- Aucune donnée sensible stockée
- Navigation sécurisée
